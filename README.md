## App Architecture and Folder Structure

#### Folder Structure

```
movies_uikit 
 ├── Resources
 │   └── Animations
 │   └── Fonts
 │   └── Assets
 ├── Application
 ├── Core
 ├── Data
 │   ├── Datasources
 │   │   └── Local Storage
 │   │   └── HTTP Client
 │   ├── Repositories
 │   │   └── Auth Repository
 │   │   └── Movies Repository
 │   │   └── User Repository
 │   ├── Models
 │   │   └── Request
 │   │   └── Response
 ├── Modules
 │   ├── Shared
 │   ├── Login
 │   ├── Launch
 │   ├── Main Tab
 │   ├── TV Shows
 │   ├── Movies
 │   ├── Library
 │   ├── Detail
 ├── Info.plist
 ├── Development.xcconfig
 └── Tests
```
